## Common Rules

> _`main` branch_ -> 1주 1발표
>
> _`intraName` branch_ -> 1일 1문제

### **_1주 1발표 Rule_**

-   주제: 풀이한 문제의 코드리뷰 혹은 다른 발표....
-   일시: 매주 화요일 2시, **개포 클러스터 4c**
-   발표자: 매주 1 ~ 2인
-   `main branch`의 해당 디렉토리에 `push`

```
e.g.)
branchName: main
│
├── Presentation
│		├── 1. dijkstra
│		│	├── imgs/
│		│	└── README.md
│		├── 2. Add Two Numbers
│		├── ...
│		└── ...
└── README.md
```

### **_1일 1문제 Rule_**

-   문제: leetcode
-   `git checkout -b intraName`
-   주말은 휴식

```
e.g.)
branchName: intraName
│
├── 1. Two Sum/
│	└── main.cpp
├── 2. Add Two Numbers/
├── ...
├── ...
```
